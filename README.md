# compilers-assignment
The practical assignment for the Compilers course [INF01147] at INF-UFRGS.

The full specification for this assignment can be found at [prof. Johann's page](http://www.inf.ufrgs.br/~johann/comp/), for the current semester (2019/2).

## To run this code:
Edit the file `test.txt` with the desired test cases. Then, run the following commands  inside the `etapa1` folder:

```
$ make
$ ./etapa1 < test.txt
```
